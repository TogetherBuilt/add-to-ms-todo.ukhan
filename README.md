# "Add to Microsoft To Do" browser extension

This extension allows you to add information about the current page (title, highlighted text, and URL) to [Microsoft To Do].

## Installation

- [Chrome web store]
- [Edge Add-ons]
- [Firefox Browser Add-ons]
